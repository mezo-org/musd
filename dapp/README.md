# MUSD Dapp

The dApp powering MUSD

## Development

### Installation

This project uses pnpm as a package manager

```bash
pnpm install
```

To run app locally:

```bash
pnpm dev
```

To create a build for production:

```bash
pnpm build
```

### Local development

During local development (no testnet / mainnet interaction), run hardhat localy in `solidity` directory

```bash
    cd ../solidity
	pnpm node
```
